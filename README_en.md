# Toolsmith MCP Server (Standalone)

Toolsmith is a **local MCP server** that can create/update/delete local MCP tool modules (`tools/*.mjs`) via tool calls, with hot-reload.

This repository is the **standalone deployment**: you can run Toolsmith separately and add it into an existing MetaMCP instance via **SSE** or **Streamable HTTP**.
